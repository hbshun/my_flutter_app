## 快速开发

* clone代码
* 安装依赖
  * 命令行：flutter pub get
  * VSCode，Android Studio： packages get
* 启动虚拟机或链接设备
* 运行程序
  * 命令行：flutter run -d [设备ID]
  * VSCode，Android Studio：启动
